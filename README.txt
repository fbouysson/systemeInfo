__________________________________ PROJET : ULTIMATE CHAT ONLINE ________________________________________

OBJECTIFS DU PROJET :

Dans le cadre de notre formation en école d'ingénieur informatique, il nous est demandé de réaliser
un chat textuel instantané. Ce dernier doit posséder un système d'authentification pour rejoindre 
des salons de discussion. Ces salons pourront être créés, rejoints ou supprimés.
Les utilisateurs auront le statut "connectés", "absent" ou "hors ligne" affichés dans les salons qu'ils
auront rejoint au préalabre.
Il y aura un historique des messages, avec la fonctionnalité de messages non lus par salon.

*********************************************************************************************************

TECHNOLOGIES UTILISEES :

Développement : Framework Symfony - HTML/CSS pour l'interface web, PHP pour l'interaction avec les bases de données,nodeJS pour certains scripts, MySQL la DB
Réseau : 5 VMs (4 windows serveur, 1 windows client), 4 routeurs, 1 switch
